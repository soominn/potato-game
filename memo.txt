- 과제 주제
Pygame + 이미지 자산을 활용한 OOP 기반 미니 게임 제작
- 필수 요구사항
1. Pygame 사용: display, event, image, time 모듈을 활용합니다.
2. 이미지 사용: 캐릭터/배경/아이템 등 최소 1개 이상 이미지(.png/.jpg)를 사용합니다.
3. 객체지향(OOP) 구조 필수
    - 권장 클래스: Game, Player, Enemy(또는 Item), Background
    - 각 클래스는 상태(속성) + 동작(메서드) 를 가집니다.
- 예시 주제
    - 피하기 게임(적/장애물과 충돌 시 종료, 시간=점수)
    - 아이템 수집 게임(제한시간 내 아이템 N개 획득)
    - 사이드 스크롤(플레이어 이동 + 스테이지 스폰)
- 제출 방법 (GitHub 버전 관리 필수)
    1. 본인 GitHub 계정에 새 저장소(Repository) 생성
        예: pygame_oop_game
        저장소 설명에 프로젝트 소개 및 주요 기능 작성
    2. 버전관리하며 프로젝트 진행
